Add New menu on the admin bar so you can access and edit the post translation fast and easy. 

== Description ==

IT REQURED WPML PLUGIN TO BE INSTALED AND ACTIVATED.

I HOPE THIS CAN HELP YOU, USE IT ON YOUR OWN RESPONSAPILITY, THANK YOU.

This Plugin adds a WPML translation menu on the admin bar on front end , so that the admin or translators can access the translation of any post and see the statue of the translations.
The plugin requires no setup and is very lightwheight.
= Features =

1. Easy access to wpml translate actions from front end.
2. Works with posts, pages, products,....
3. Works out of the box - no setup needed.
4. Very lightweight plugin.
5. Show the translation statu of each language on the front end..


== Installation ==

1. Upload the redolance-wpml-easy-access folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! You should now be able to access your post translation actions from the admin bar on front end.
